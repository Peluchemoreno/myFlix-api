This is the documentation for interfacing with the myFlix API.

GET /movies

Returns a full list of movies.

Output:

{
  "_id": "abc123",
  "Title": "Inception",
  "Description": "A thief who steals corporate secrets...",
  "Genre": { "Name": "Sci-Fi", "Description": "..." },
  "Director": { "Name": "Christopher Nolan", "Bio": "..." }
}

GET /movies/:title

Get information about a movie by its title.

Output:

{
  "_id": "abc123",
  "Title": "Inception",
  ...
}

GET /genres/:name

Returns information about a specific genre.

Output:

{
  "Name": "Sci-Fi",
  "Description": "Fiction based on imagined future technologies..."
}

GET /directors/:name

Returns information about a specific director.

Output:

{
  "Name": "Christopher Nolan",
  "Bio": "...",
  "BirthYear": "1970"
}

POST /users

Create a new user account.

Input:

{
  "username": "johndoe",
  "password": "mypassword",
  "email": "john@example.com",
  "birthday": "1990-01-01"
}

Output:

{
  "_id": "user123",
  "username": "johndoe",
  "email": "john@example.com",
  "favoriteMovies": []
}
